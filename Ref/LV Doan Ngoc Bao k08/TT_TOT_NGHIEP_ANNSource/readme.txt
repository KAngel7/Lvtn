Cấu trúc file network.xml - đặc tả một mạng neuron để lưu trữ xuống file
- Là một file xml với root là <network>
- Trong đó:
	numInputNodes, numHiddenNodes, numOutputNodes: số lượng node trong các lớp tương ứng
	nhánh InputNodes chứa các node input. Trong nhánh này gồm các node với các tên là 1,2,... <tăng theo số lượng nodes>. Quy tắc tương tự đối với các nhánh còn lại
